An ultra-high-performance, industry-standard playground for practicing Fluxy DSL and standard Flutter code.

## Key Features
- **Hybrid Framework Engine**: Seamlessly switch between Fluxy and Flutter modes.
- **Industry Layout**: Professional vertically-split editor and preview area (Expo Snack style).
- **Advanced Debugging**: Built-in Widget Inspector and State Memory viewer.
- **Stability Kernel**: Real-time monitoring of layout, state, and rendering performance.
- **Zero-Static Workspace**: Fully dynamic snippets library and terminal console.

## Getting Started
1. Run the app: `flutter run`
2. Select a snippet from the side explorer.
3. Start practicing code in the editor!

## Tech Stack
- **Framework**: Flutter with Fluxy signals & DSL.
- **State Management**: Fluxy Signals.
- **Engine**: Dynamic pattern-matching simulation engine.
